To change the speed of the extruder when it is executing g-code, we could modify the default setting for DEFAULT_AXIS_STEPS_PER_UNIT for the ‘E’ axis, in this case a value of 500 below. The range of numbers on the plastic knob is 1 to 10. If that could be mapped to the range of 200 to 800 in the ‘E’ axis steps per unit, it would probably be sufficient. 




// default settings

#define DEFAULT_AXIS_STEPS_PER_UNIT   {100,100,1600,500}  // default steps per unit for Mini Metal Maker V1.2
#define DEFAULT_MAX_FEEDRATE          {300, 300, 2, 45}    // (mm/sec)
#define DEFAULT_MAX_ACCELERATION      {9000,9000,100,10000}    // X, Y, Z, E maximum start speed for accelerated moves. E default values are good for Skeinforge 40+, for older versions raise them a lot.
